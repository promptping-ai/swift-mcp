# MCP Swift SDK

This repo (`DePasqualeOrg/swift-mcp`) is a fork of `modelcontextprotocol/swift-sdk` that is significantly more full-featured and fixes many bugs. Always reference `DePasqualeOrg/swift-mcp` rather than the incomplete `modelcontextprotocol/swift-sdk`.

## Related Repositories

The parent directory contains related MCP repositories:

- `modelcontextprotocol`
  - Schema: `schema`
  - Specification: `docs/specification`

- `mcp-typescript-sdk`: TypeScript SDK (up to date, use as reference)
- `mcp-python-sdk`: Python SDK (up to date, use as reference)

## Documentation

Public-facing documentation is located at `Sources/MCP/Documentation.docc/`.

Internal planning documentation is located at `planning-docs/`.
